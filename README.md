# Pastel Bubble Playlist  
감성적인 파스텔 톤과 비눗방울 모션을 활용한 **3곡 플레이리스트 웹페이지**

본 프로젝트는 HTML/CSS 기반으로 제작된 정적 웹페이지로,  
사용자 4명이 각자 좋아하는 세 곡을 감성적인 인터페이스 안에서 소개하는 웹 프로젝트입니다.  
파스텔톤 그라데이션, 반투명 글래스모피즘 카드, 비눗방울 배경 애니메이션 등을 적용하여  
부드럽고 몽글한 감성 UI/UX를 구현하였습니다.


## 프로젝트 목적
- 단순한 음악 소개를 넘어 감성적인 분위기 전달을 목표로 함  
- 웹 디자인 요소(그라데이션, 블러, Glow, 부드러운 호버 효과 등)를 직접 구현하며  
  HTML/CSS UI 제작 역량 강화  
- GitHub Pages를 활용하여 **실제 웹사이트처럼 배포하는 경험** 확보  
- Git 기반 버전 관리 및 프로젝트 기록 자동화


## 소개하는 12곡
1. **Karencici – 愛你但說不出口**  
2. **HONNE – Warm On A Cold Night**  
3. **검정치마 – 섬 (Queen of Diamonds)**  
4. **HANRORO - 비틀비틀 짝짜꿍**
5. **UmYull - 피차일반**
6. **DAY6 - 날아라! 드림라이더**
7. **라붐 - 상상더하기**
8. **프로미스나인 - 하얀 기다림**
9. **레드벨벳-day1**


각 곡은 감성적인 텍스트 설명, 태그, 분위기 레이블과 함께 카드 형태로 배치되어 있으며,  
사용자가 특정 곡의 느낌을 직관적으로 이해할 수 있도록 디자인되었습니다.

## 프로젝트 기여 및 필요성
이 프로젝트에서 구현된 웹페이지는 단순한 음악 목록이 아니라,

- **사용자 경험(UX)을 강화한 가벼운 감성형 UI 사례**
- **직접 디자인한 애니메이션 요소(비눗방울 float)**
- **반응형 레이아웃 구성**
- **GitHub 기반 배포 → 실제 웹 서비스와 동일한 구조**

이라는 점에서 학습적 가치와 실용성이 있습니다.

또한, GitHub Pages를 활용해 정적 웹사이트 제작 → 개발 → 배포 과정을  
혼자서 end-to-end로 경험할 수 있어 **개발 역량을 크게 향상시키는 프로젝트**입니다.


## 팀원 및 역할 

- **UI/UX 디자인**: 파스텔/비눗방울 컨셉 설정, 레이아웃 구성  
- **프론트엔드 구현**: HTML 구조 설계, CSS 스타일링, 애니메이션 적용  
- **배포 및 버전 관리**: Git, GitHub Repository 관리, GitHub Pages 배포
